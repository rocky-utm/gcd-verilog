### GCD Calculator

#### Requirements
- gcc
- g++
- make
- cmake
- [verilator](https://www.veripool.org/verilator/)
- [gtkwave](https://gtkwave.sourceforge.net/)

#### Usage
```bash
$ make cmake
$ make
$ make show
```
